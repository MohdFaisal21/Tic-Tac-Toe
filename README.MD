# Tic Tac Toe Game

A simple Tic Tac Toe game built using HTML, CSS and JS

This is a simple Tic Tac Toe game where two players take turns marking X and O.
The game checks for a winner or a draw after each move.


## Features
- Two-player gameplay
- Interactive game board
- Win and draw detection
- Restart game option

## Technologies Used
- HTML
- CSS
- JavaScript
